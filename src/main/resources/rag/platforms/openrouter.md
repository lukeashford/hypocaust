## Llama 4 Maverick

- **owner**: meta-llama
- **id**: llama-4-maverick
- **tier**: powerful
- **input**: TEXT
- **output**: TEXT

### Description

Llama 4 Maverick is a high-performance TEXT generation model from Meta, excelling at creative
writing,
poetry, long-form storytelling, brainstorming, and nuanced language tasks. It is a frontier-scale
model with strong instruction following, a large context window, and excellent multilingual support.
Best for complex creative tasks where high-quality prose and style are required.

### Best Practices

- Use detailed system prompts to set tone, style, and constraints for the best creative output.
- For long-form content, break generation into sections with clear structural guidance.
- Temperature 0.7-0.9 for creative tasks; 0.2-0.4 for factual/structured output.
