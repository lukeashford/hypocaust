## Llama 4 Maverick

- **owner**: meta-llama
- **id**: llama-4-maverick
- **tier**: powerful

### Description

Llama 4 Maverick is a high-performance TEXT generation model from Meta, excelling at creative writing,
long-form content generation, brainstorming, and nuanced language tasks. Strong instruction following
with a large context window and excellent multilingual support.

### Best Practices

- Use detailed system prompts to set tone, style, and constraints for the best creative output.
- For long-form content, break generation into sections with clear structural guidance.
- Temperature 0.7-0.9 for creative tasks; 0.2-0.4 for factual/structured output.

## DeepSeek V3.1

- **owner**: deepseek
- **id**: deepseek-chat-v3-0324
- **tier**: balanced

### Description

DeepSeek V3.1 is a balanced TEXT generation model optimized for reasoning, code generation, and
analytical writing. Strong performance on structured tasks, technical content, and step-by-step
explanations with competitive cost efficiency.

### Best Practices

- Leverage its reasoning strength with chain-of-thought prompting for complex analytical tasks.
- For code generation, provide clear specifications and example inputs/outputs.
- Works well with system prompts that define output format expectations.

## Qwen3 235B

- **owner**: qwen
- **id**: qwen3-235b
- **tier**: powerful

### Description

Qwen3 235B is a large-scale TEXT generation model with exceptional capability in creative writing,
translation, summarization, and complex reasoning tasks. Its large parameter count enables nuanced
understanding and generation across diverse domains and languages.

### Best Practices

- Ideal for tasks requiring deep reasoning or multi-step analysis; use structured prompts.
- For creative tasks, provide style references and tone guidance in the system prompt.
- Supports very long context; useful for document analysis and summarization workflows.
