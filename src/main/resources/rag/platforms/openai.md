### Description

GPT-4o ("omni") is a versatile, high-performance model designed for speed and efficiency without
sacrificing intelligence. It excels at a wide range of tasks including coding, translation, and

### Description

GPT-4o mini is a highly cost-efficient, small-scale model designed for high-volume, low-latency
