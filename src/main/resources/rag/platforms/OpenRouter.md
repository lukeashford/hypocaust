# OpenRouter

## GPT-4.1 Mini

Summary: Lightweight, cost-efficient LLM proxying OpenAI-compatible APIs via OpenRouter marketplace.

- Strengths: affordability, wide vendor coverage, unified API.
- Weaknesses: quality varies by upstream provider.
- Notes: Set provider order and fallbacks in headers.

## Llama-3.1-8B

Summary: Open-source 8B parameter model with strong reasoning for its size; great for on-device or
edge use.

- Strengths: latency, controllability, transparent licensing.
- Weaknesses: limited world knowledge vs frontier models.
- Notes: Prefer short system prompts; leverage tools for retrieval.
